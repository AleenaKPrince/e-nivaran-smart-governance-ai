# E-Nivaran Smart Governance AI Platform

## Overview
E-Nivaran is an AI-powered smart governance and grievance redressal platform designed to improve complaint management and citizen-government interaction. The platform uses AI, NLP, and automated routing mechanisms to classify, prioritize, and manage public grievances efficiently.

## Features
- AI-based complaint classification
- Smart complaint routing system
- Multilingual chatbot support
- Role-based dashboards
- Complaint tracking and management
- Department analytics and reporting
- Authentication and authorization
- Staff workload management
